UD1. Introdución a Javascript

Ejercicio 0 (Inventado por el profesor)

Calcula la secuencia de fibonacci atendiendo a la formula:

  • fib(n) = fib(n-1) + fib(n-2)
  • fib(0) = 0
  • fib(1) = 1

Ejercicio 1 (Inventado por el profesor)

Calcula el factorial de un numero (n!)

  • fact(n) = n * fact(n-1)
  • fact(0) = 1

Ejercicio 2 (Inventado por el profesor)

Desarrolla un programa que le pida al usuario una cantidad de dias en valor entero y el programa debe devolver cuantas horas, minutos y segundos son

Ejercicio 3 (Inventado por el profesor)

Realiza un programa capaz de resolver una ecuacion de segundo grado. Para ello se solicita al usuario que introduzca los coeficientes a, b y c y devolvera las soluciones (si las hay) de la ecuacion.
